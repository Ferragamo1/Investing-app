How to set this up in Robinhood

Uses fractional shares and recurring buys
  1. Search VOO → tap Buy → switch to Amount → set Recurring → Weekly → enter $12.
  2. Search NVDA → set a recurring weekly buy for $5.
  3. At the start of each new month, set or edit a recurring weekly buy of $3 for the listed rotation ticker for that month.
  4. Keep deposits at or above $20 per week so the script always has funds to run.
